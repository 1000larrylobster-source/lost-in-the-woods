# Lost in the Woods

You wake up off-trail with a dead phone and half a granola bar.
Someone finds you. A drone, a radio ping, a stranger with a mesh node.
You do not forget that.
So you learn the hardware. Radios first, then drones, then a sensor grid.
You build the network that found you, piece by piece, on a mountain that does not care.
And then you become the one who finds people.

## The core loop

**Get found. Become the finder.**

1. Survive as a lost hiker. Get rescued by the network.
2. Join the search side. Salvage parts, build gear, expand coverage.
3. Find other lost hikers. Catch fires while they are still one tree.

Search-and-rescue meets base-builder. The base is a communications network stretched over wilderness.

## The tech tree

Every rung is real hardware you could actually build. That is the rule.

```
bare hands  ->  radio  ->  drone  ->  mesh network  ->  sensor grid
```

- **Bare hands**: signal mirror, whistle, fire. Act 1 survival.
- **Radio**: handheld transceiver, then a LoRa beacon you solder yourself.
- **Drone**: Pi Zero flight controller, scavenged motors, a camera that sees heat.
- **Mesh network**: solar LoRa nodes on ridgelines. Coverage is territory.
- **Sensor grid**: thermal cams, smoke sensors, trail counters. The forest gets a nervous system.
