# itch page description — paste-ready (v1.0, $1)

*You wake lost. Build the forest's eyes. The ones you save stay.*

**Lost in the Woods** is a survival wildfire-watch game in one HTML file. Assemble solar
towers from parts you haul on your back, fly drones that hunt the first curl of smoke,
feed yourself, tend your machines, and turn rescued strangers into a community of
watchers — across ten regions of one Range.

Every mechanic is real: the tower you build in the game is a real Raspberry Pi
fire-detection node, fictionalized. The game and the hardware are being built in public,
in one motion.

**The network is the game.** Panels foul. Batteries brown out at night. Rotors wear.
Storms take antennas. A fire you miss burns forest that stays burned — the save file
*is* the forest.

**Survival that serves the watch.** Hunger, thirst, warmth, stamina — and the same
drones that hunt smoke spot the elk you stalk downwind for food.

**Ten regions, one year.** Living wind, night fires, storms, snow, drought embers, dry
lightning, a closed parts economy, and a fire-season finale that tests everything and
everyone you've built.

### What you get
- The full game: one self-contained HTML file, plays in any browser, desktop + touch
- 10 regions with distinct palettes, tempers, and one new mechanic each
- 8 named recruits with real jobs; 7 easter eggs nobody will mark on a map
- All art and audio generated by code — zero assets, instant load
- Every update included, forever. Built in public — the devlogs are in the repo.
